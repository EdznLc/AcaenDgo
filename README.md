# AcaenDgo — Mapa de negocios de Durango

Proyecto frontend mínimo en React con un componente de mapa usando Leaflet y OpenStreetMap.

Run locally:

```bash
cd AcaenDgo
npm install
npm run dev
```

Abre luego:

http://localhost:5173/

Notas:
- La app intenta cargar comercios desde `http://localhost:4000/api/comercios`.
- Si el backend no está en ejecución, el mapa mostrará un mensaje de error en lugar de datos.
- No hay registro de usuarios en esta versión de mapa.
